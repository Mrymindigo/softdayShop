# Features

This document describes all planned features for the SoftDay online store.

---

# User Features

## Browse Products ✅ (MVP)

Users can browse all available products.

Features:
- Product grid
- Product cards
- Pagination
- Responsive layout

---

## Product Details ✅ (MVP)

Each product has its own detail page.

Features:
- Product images
- Image gallery
- Product title
- Description
- Price
- Discount price
- Available colors
- Available sizes
- Stock status
- Add to cart button

---

## Categories ✅ (MVP)

Users can browse products by category.

Features:
- Category list
- Category page
- Category filtering

---

## Search ✅ (MVP)

Users can search products.

Version 1:
- Search by product title

Future:
- Search by description
- Search suggestions

---

## Filters ✅ (MVP)

Users can filter products.

Filters:
- Category
- Price
- Color
- Size

Future:
- Availability
- Best Seller
- New Products

---

## Shopping Cart ✅ (MVP)

Users can manage their cart.

Features:
- Add product
- Remove product
- Increase quantity
- Decrease quantity
- Total price calculation

---

## Checkout ✅ (MVP)

Users can place orders.

Fields:
- Full Name
- Phone Number
- Address
- Notes

Future:
- Online payment gateway

---

## Order Confirmation ✅ (MVP)

Show successful order message after checkout.

---
